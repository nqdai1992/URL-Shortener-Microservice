Freecodecamp API : URL Shortener Microservice

This API create short URL redirect to your link that you pass in URL as parameter

Example

When you go to https://create-short-url.herokuapp.com/new/https://www.freecodecamp.com, API will give you a shorter link https://create-short-url.herokuapp.com/new/123 that redirect to https://www.freecodecamp.com
